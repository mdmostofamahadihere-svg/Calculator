## Installation

1.  Navigate to the project directory:
    ```bash
    cd "Calculator App"
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

## Deployment

To deploy this app (e.g., to Vercel or Netlify):

1.  Build the project:
    ```bash
    npm run build
    ```
2.  Upload the `dist` folder to your hosting provider or connect your repository to Vercel/Netlify for automatic deployments.
